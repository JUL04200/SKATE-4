SCHÉMA · SCAFFOLD LAUNCH GLITCH

Launch Glitch
au Scaffold

Exploit physique exploitant la hitbox mal calculée du board en porte-à-faux. Quand le joueur approche avec le bon timing, le moteur génère une force d'impulsion anormale — catapulte instantanée vers n'importe quel point de la map.

DIFFICULTÉ
4/10
  1. 01
    Déploie un Scaffold

    N'importe où sur la map — surface plane recommandée.

  2. 02
    Pose un Manual Pod sur le dessus

    Near the edge pour que le board puisse déborder dans le vide.

  3. 03
    Board au bord (dans le vide)

    La hitbox qui pend crée le trigger point de l'exploit.

  4. 04
    Run → ✕ SAUT juste avant le contact

    Frame window ~2-3 frames. Timing précis mais learnable rapidement.

PRO TIP

Pre-place un scaffold avant un challenge pour traverser la map en secondes. Combine avec le Speed Glitch pour une vitesse extrême.

BUGS SAISON 4

Bugs actifs — statuts & workarounds

🧍

Player Model Glitch

● ACTIF
PS4PS5

Le modèle du joueur se déforme en stance Switch et Fakie. Bug purement visuel — le gameplay n'est pas affecté.

WORKAROUND

Aucun fix permanent. Switch rapidement de stance pour reset l'affichage temporairement.

🚗

Voitures Invisibles

● ACTIF
PS4PS5

Toggle "World Traffic" → certaines voitures disparaissent visuellement mais gardent leur hitbox et leur comportement physique.

WORKAROUND

Toggle "World Traffic" deux fois de suite → force le rechargement des assets visuels.

⬛

Écran Noir Challenge

● PS4 ONLY
PS4

Au lancement d'un challenge sur PS4, écran noir complet. Le jeu freeze — aucune interaction possible.

WORKAROUND

Seul le reboot complet fonctionne : bouton PS → Quitter le jeu → Relancer.

GLOSSAIRE

Tous les termes techniques expliqués

Hippy Jump

Trick où le rider saute par-dessus un obstacle pendant que le board passe en dessous — base du Spread Eagle glitch.

Spread Eagle

Pose aérienne bras+jambes écartés en X, hors du board. Trigger du speed glitch dans skate.

Switch

Rider qui skate dans l'orientation opposée à sa stance naturelle (regular ↔ goofy).

Fakie

Rider qui roule en arrière dans sa stance naturelle — différent du switch.

No Foot Air

Trick aérien où le rider lâche le board. Résultat si le Spread Eagle est fait SUR le board.

Manual Pod

Objet placeable permettant les manuels. Composant clé du Scaffold Launch Glitch.

Scaffold

Structure placeable dans le créateur de spots. Sert de rampe dans le Launch Glitch.

Double Grind

2 tricks de grind en succession ultra-rapide sur un même rail. Glitch devenu feature officielle.
